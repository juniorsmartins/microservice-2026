### Introdução: 

```
O Flyway estende o DevOps para seus bancos de dados para acelerar a entrega de software e 
garantir código de qualidade para que você possa implantar com confiança. Do controle de 
versão à entrega contínua, o Flyway baseia-se em processos de entrega de aplicativos para 
automatizar implantações de banco de dados. 



```
